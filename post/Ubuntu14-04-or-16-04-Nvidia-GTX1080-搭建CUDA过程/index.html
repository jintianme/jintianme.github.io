Ubuntu14.04 or 16.04 Nvidia GTX1080 搭建CUDA过程

   2016-12-17  

一定要ssh远程登录 关闭开启图形界面

sudo service lightdm start

到后面可能换会出现 NVIDIA driver install - Error: Unable to find the kernel source tree的错误，解决办法 sudo update-grub uname -r 会显示当前内核

cd /usr/src 会有kernel的位置

卸载Nvidia驱动 sudo nvidia-uninstall
